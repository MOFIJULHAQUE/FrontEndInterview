Box-sizing is a CSS property that allows developers to define how the browser should calculate the size of an element. It is used to set whether the total width and height of an element should include padding and borders, or not. By default, the box-sizing property is set to content-box, which means that the width and height of an element will only 
include its content, and not its padding or borders.
